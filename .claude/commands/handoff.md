End-of-sitting ritual. Do all of this:
1. Stage and commit ALL current work on the current branch (including any
   edits made in docs/ via Obsidian) with a clear message; push. If there
   is nothing to commit, continue to step 2.
2. Create or update docs/HANDOFF.md with: the current branch name, what is
   done, what is half-finished, the exact next steps, and any gotchas
   discovered this session. Keep it under 20 lines.
3. Commit ("chore: handoff note") and push that too.
4. Confirm to me that everything is pushed and it is safe to walk away.
