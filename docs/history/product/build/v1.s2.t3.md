## Story

Roam got its database: a fresh Supabase Postgres project in us-east-1,
co-located with the Vercel build region per the D-005 doctrine, with a
server-side-only memoized client guarded by the server-only package so
database credentials can never reach the browser. Auth session
persistence stays off until accounts arrive.
