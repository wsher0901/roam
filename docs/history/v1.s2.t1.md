## Story

The docs-only repo became a runnable app: Next.js 16 + TypeScript +
React 19 (App Router, Turbopack), Tailwind v4 + shadcn/ui with one
vendored Button to prove the pipeline, and the isolated engine/
directory beside the app — with an ESLint rule enforcing that engine
files import no app, React, or Next code — plus Vitest's first passing
smoke test.
